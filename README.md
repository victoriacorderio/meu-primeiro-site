📄 Meu Primeiro Site
- Este repositório contém o código do primeiro site que desenvolvi em HTML, como parte do meu aprendizado em Desenvolvimento Front-End.
O objetivo foi praticar estruturação de conteúdo, semântica HTML, tabelas e links, além de criar uma página simples para apresentar informações sobre mim.

✨ Funcionalidades
- Apresentação pessoal com título e descrição.
- Seção de formações listadas em tabela.
- Hard Skills e Soft Skills organizadas em tabelas.
- Links clicáveis para minhas redes profissionais: LinkedIn, Behance e Instagram
- Mensagem de boas-vindas via alert() em JavaScript.

🛠️ Tecnologias utilizadas
- HTML5 – estrutura e marcação.
- CSS (básico, inline e via atributos) – bordas de tabela (em breve será separado para um arquivo próprio).
- JavaScript – para interação inicial (alerta de boas-vindas).
